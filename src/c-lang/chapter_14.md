# chapter 14 ファイル入出力

## 14.1 ファイルの種類

プログラムは、画面やキーボードだけでなく、ファイルともやり取りします。  
ファイル入出力を覚えると、設定の保存、ログの記録、データの読み込みができるようになります。

ファイルには、文字として読む**テキストファイル**と、バイト列として扱う**バイナリファイル**があります。  
まずはテキストファイルから始めると理解しやすいでしょう。

## 14.2 ファイルの読み書きの基本

C では、ファイルを `FILE *` で扱います。

```c
FILE *fp = fopen("data.txt", "r");
if (fp == NULL) {
    printf("ファイルを開けませんでした\n");
}
```

`fopen` の第 2 引数にはモードを書きます。  
`"r"` は読み込み、`"w"` は書き込み、`"a"` は追加書き込みです。

## 14.3 1文字の読み書き

1 文字ずつ読むには `fgetc`、書くには `fputc` を使えます。

```c
int ch;
while ((ch = fgetc(fp)) != EOF) {
    putchar(ch);
}
```

`EOF` はファイルの終わりを表します。  
文字 1 つではなく `int` で受けるのは、`EOF` と区別するためです。

## 14.4 まとまった文字の読み書き

行単位で扱うなら `fgets` と `fputs` が便利です。

```c
char line[256];
if (fgets(line, sizeof(line), fp) != NULL) {
    printf("%s", line);
}
```

`fgets` は、バッファサイズを指定できるので、読みすぎを防ぎやすいのが利点です。

## 14.5 サイズ指定による読み書き

バイナリデータを読むときは `fread` と `fwrite` を使います。

```c
size_t n = fread(buffer, 1, sizeof(buffer), fp);
```

この種の関数は、読み書きの「単位」と「回数」を意識する必要があります。  
初学者はまず、テキスト入出力をしっかり理解し、そのあとでバイナリへ進むとよいでしょう。

## 14.6 ランダムアクセス

`fseek` を使うと、ファイルの途中へ移動できます。  
これは、ファイルを前から順に読むだけでなく、好きな場所を読むときに使います。

```c
fseek(fp, 0, SEEK_END);
long size = ftell(fp);
```

## 14.7 ファイル自体の操作

ファイルは開くだけでなく、閉じる必要があります。

```c
fclose(fp);
```

`remove` や `rename` を使うと、ファイルの削除や名前変更もできます。  
ファイル操作は便利ですが、失敗の可能性もあるので、戻り値の確認を習慣にしましょう。

## 14.8 第14章のまとめ

- ファイルは `FILE *` で扱います
- `fopen` と `fclose` は基本中の基本です
- 行単位なら `fgets`、1 文字なら `fgetc` が便利です
- バイナリとテキストでは考え方が少し違います
