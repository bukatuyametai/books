# chapter 2 変数と型

## 2.1 変数宣言の文

変数は、**値を入れておく名前つきの箱**です。  
C では、箱の種類を最初に決めてから使います。この「種類」が型です。

```c
int score;
double pi;
char grade;
```

このように書くと、`score` は整数を入れる箱、`pi` は小数を入れる箱、`grade` は 1 文字を入れる箱になります。  
C では、型を決めることがとても重要です。なぜなら、型によって使える値の範囲や計算のしかたが変わるからです。

宣言だけでは中身は決まりません。使う前には、できれば値を入れておくのが安全です。

```c
int score = 100;
double temperature = 23.5;
char letter = 'A';
```

## 2.2 代表的な型

最初に覚えたい代表的な型は次のとおりです。

- `int`: 整数
- `char`: 文字 1 つ
- `float`: 単精度浮動小数点数
- `double`: 倍精度浮動小数点数

整数には、`short`、`long`、`long long`、`unsigned` などの派生もあります。  
ただし初学者は、まず `int` と `double` を中心に使えば十分です。必要になったときに、より大きい型や符号なし型を学びましょう。

文字型 `char` は、見た目は文字を扱うための型ですが、内部的には数値として扱われます。`'A'` と `"A"` は似ていますが同じではありません。`'A'` は 1 文字、`"A"` は文字列です。この違いは後で何度も出てきます。

## 2.3 初期化と定数

変数は、宣言と同時に初期化するのが基本です。

```c
int count = 0;
double price = 1980.0;
char mark = '*';
```

定数には `const` を使います。  
値が変わらないことを明示しておくと、コードの読みやすさが上がります。

```c
const double TAX_RATE = 0.10;
```

`const` をつけた変数は、あとから別の値を代入できません。  
「途中で変えてはいけない値」を `const` にしておくと、思わぬ書き換えを防げます。

## 2.4 文字列の取り扱い

C では、文字列は少し特別です。  
多くの言語のように「文字列型」が最初からひとつ用意されているわけではありません。C では通常、**文字の並びを配列として扱います**。

```c
char name[] = "Taro";
```

ここで `name` は、`'T'`、`'a'`、`'r'`、`'o'`、終端を表す `'\0'` を並べた配列です。  
この終端文字 `'\0'` は非常に重要です。C の文字列は、`'\0'` を見つけるまでを文字列として読みます。

文字列については、後の章でくわしく学びます。今は「C の文字列は、文字の配列として表現される」と覚えておけば十分です。

## 2.5 第2章のまとめ

- 変数は値を入れておく名前つきの場所です
- 型は、その変数が扱う値の種類を決めます
- 初期化は、変数に最初の値を入れることです
- C の文字列は、`'\0'` で終わる文字の並びです
