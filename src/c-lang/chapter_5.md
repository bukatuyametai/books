# chapter 5 制御構文のバリエーション

## 5.1 分岐構文のバリエーション

`if` だけで十分なことも多いですが、条件が増えると書き方に工夫が必要になります。

```c
if (x > 0) {
    printf("正の数です\n");
} else if (x < 0) {
    printf("負の数です\n");
} else {
    printf("0 です\n");
}
```

このように、`else if` をつなげることで、複数の条件を順番に判定できます。  
また、値に応じて処理を分けるときは `switch` が便利です。

```c
switch (grade) {
    case 'A':
        printf("とても良い\n");
        break;
    case 'B':
        printf("良い\n");
        break;
    default:
        printf("次はがんばろう\n");
        break;
}
```

`switch` は、同じ変数の値によって複数の分岐をきれいに書けます。

## 5.2 繰り返し構文のバリエーション

繰り返しには `while` と `for` のほかに `do ... while` があります。  
`do ... while` は、**少なくとも 1 回は実行される**のが特徴です。

```c
int n = 0;
do {
    printf("%d\n", n);
    n++;
} while (n < 3);
```

`for` は、回数が決まっているくり返しに向いています。  
`while` は、条件が満たされている間くり返すときに向いています。

```c
for (int i = 0; i < 5; i++) {
    printf("%d\n", i);
}
```

`break` はループを抜け、`continue` はその回の残りを飛ばして次へ進みます。  
ただし、使いすぎると流れが見えにくくなるので、最初は控えめに使うのがよいでしょう。

## 5.3 制御構文の応用

複数のループを重ねると、表やグリッドのような処理が書けます。

```c
for (int y = 1; y <= 3; y++) {
    for (int x = 1; x <= 3; x++) {
        printf("(%d,%d) ", x, y);
    }
    printf("\n");
}
```

これは、後の章で作るゲームや表計算のような処理の土台になります。

## 5.4 第5章のまとめ

- `else if` で条件を順に判定できます
- `switch` は値ごとの分岐に向いています
- `do ... while` は 1 回は必ず実行されます
- `break` と `continue` はループの流れを変えます
