# C言語入門

**はじめてのプログラミングから実用開発まで**


この本は、**はじめてプログラミングに触れる人**を主な対象にした C 言語の入門書です。  
「C は難しい」と言われることがありますが、難しさの多くは、文法そのものよりも、メモリ・ポインタ・コンパイル・複数ファイル開発といった周辺知識にあります。そこで本書では、最初はなるべく小さな世界から始め、少しずつ視野を広げながら、最後には実際の開発で役立つ道具までたどり着けるように構成しました。

本書の方針は次のとおりです。

- 初学者がつまずきやすいところを、急がず丁寧に説明する
- 古い慣習よりも、**今の標準 C で自然な書き方**を優先する
- ただし、いきなり高度な話に寄りすぎず、最初は「動く」「読める」「直せる」を重視する
- たくさんの小さな例題を通して、手を動かしながら覚えられるようにする

C 言語は、今も広く使われています。OS、組み込み機器、通信、ゲームエンジン、高速処理が必要な場面などで、C の知識は今なお価値があります。本書では、**ただ古い言語を学ぶ**のではなく、現在の標準に沿った C の見方を身につけることを目指します。C の現行標準は C23 です。初学者向けの本としては、まずは基本をしっかり理解し、その上で新しい規格の存在を知っておくのがよいでしょう。

## この本の構成

本書は大きく分けて2つのパートで構成されています。

- **C言語編**: 基本から高度なメモリ管理、複数ファイル開発までをじっくり学びます。
- **Python編**: 初心者に優しいPythonを使って、プログラミングの楽しさと実用的なアプリ作りを体験します。

## この本の読み方

最初から順番に読むのが基本ですが、必要に応じて章を行き来してもかまいません。  
特におすすめなのは、次の順番です。

### C言語編の進め方
1. Chapter 0 から Chapter 5 で、文法と制御構文に慣れる
2. Chapter 6 から Chapter 8 で、配列・構造体・関数を覚える
3. Chapter 9 以降で、C らしい世界であるポインタ・メモリ・文字列・ファイルへ進む
4. Chapter 13 以降で、複数ファイルやツールを使った開発に進む

### Python編の進め方
1. 第1章から第5章で、基本的な文法をマスターする
2. 第6章以降で、実用的なライブラリやオブジェクト指向、ファイル操作に進む
3. 最後はRPG製作プロジェクトに挑戦します！

各章の最後には、要点のまとめと練習問題を置いてあります。練習問題は、答えを読むだけでなく、できれば自分でコードを書いて確かめてください。C は「知っている」だけでは身につきません。**書いて、コンパイルして、失敗して、直す**ことがとても大切です。
