### 1-1. プログラミングとは何か

プログラミングは、コンピュータにやってほしいことを、順番とルールがわかる形で書くことです。人に頼むときは少し曖昧でも通じることがありますが、コンピュータは曖昧さが苦手です。だからこそ、手順を小さく分けて、はっきり書く練習が大切です。

Python は次の理由で初心者に向いています。

- 読みやすい
- 書く量が比較的少ない
- Web、AI、データ分析、自動化、ゲームづくりまで幅広く使える

> ミニ体験  
> 電卓に向かって「10 と 5 を足して、その結果を 2 で割って」とお願いすると考えてみましょう。人は途中を補って理解できますが、コンピュータにはその補助がありません。命令を順番に並べること自体が、プログラミングの基本です。

### 1-2. 学習の準備をしよう

まずは Python 本体と、コードを書くためのエディタを用意します。おすすめは VS Code です。Python の拡張機能を入れると、色分けや補完が使えて見通しがよくなります。

```bash
python --version
# 環境によっては python3 --version
```
> チェック課題  
> ターミナルで Python のバージョンを表示できれば準備完了です。うまくいかなくても焦らなくて大丈夫です。ここでつまずくのは、とてもよくあることです。

### 1-3. はじめてのプログラム

```python
print("Hello, World!")
print("こんにちは、Python")
```
`print()` は、画面に文字を出すための命令です。最初はこれだけで十分です。出力できたなら、もう立派なプログラムです。

### エラーが出ても慌てない
エラーは「失敗」ではなく、「どこを見直せばよいかを教えてくれるヒント」です。最初は、エラーを見て止まるのではなく、エラーを読んで直す練習をしましょう。

> 章末ミニゲーム  
> **自己紹介メーカー** を作りましょう。名前、好きな食べ物、好きな色を表示するだけの短いプログラムです。

```python
print("名前: Taro")
print("好きな食べ物: カレー")
print("好きな色: 青")
```
