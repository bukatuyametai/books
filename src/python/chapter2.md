# 第2章 変数とデータ型

## この章のゴール
- 値を変数に入れられる
- 数字と文字列の違いがわかる
- 入力を受け取って使える

### 2-1. 変数はデータの箱

変数は、値を入れておく名前付きの箱です。あとで同じ値を何度も使いたいときに便利です。

```python
name = "Taro"
age = 20

print(name)
print(age)
```
変数名は、意味がわかる名前にすると読みやすくなります。`x` や `a` でも動きますが、最初のうちは `name`、`age`、`score` のような名前がよいです。

### 2-2. 数字と文字列

```python
print(10)
print(3.14)
print("Python")
```
数字は計算できますが、文字列は文字として扱います。ここを分けて考えることがとても大切です。

```python
print(10 + 5)
print(10 / 3)
print(10 // 3)
print(10 % 3)
```
- `+` 足す
- `-` 引く
- `*` 掛ける
- `/` 割る
- `//` 商だけを取る
- `%` 余りを取る

### 2-3. 入力して使う

```python
name = input("名前を入力してください: ")
print("こんにちは、" + name + "さん")
```
`input()` は、キーボードから文字を受け取ります。受け取った値は文字列として扱われるので、数字として使いたいときは変換が必要です。

```python
age = int(input("年齢を入力してください: "))
print("来年は", age + 1, "歳です")
```
### 2-4. 文字列の連結と注意点

```python
item = "ポーション"
count = 3

print("持っているのは " + item)
print("数は " + str(count))
print("数は", count)
```
数字と文字列をそのまま `+` でつなぐと、エラーになることがあります。迷ったら、`print()` のカンマ区切りを使うと安全です。
